# BINANCE FINANCE DATA

Finance Data is a Python library made which utilizes the Binance API to collect and download candlestick data of currencies into csv file. 

## Supported Intervals

### Minute
1) 1_Min
2) 3_Min
3) 5_Min
4) 15_Min
5) 30_Min

### Hour
1) 1_Hour
2) 2_Hour
3) 4_Hour
4) 6_Hour
5) 8_Hour
6) 12_Hour

### Day
1) 1_Day
2) 3_Day

### Week
1) 1_Week

### Month
1) 1_Month
